# Boykisser-DDoS
I made This DDoS Tool with batman as a Joke and is only used for cybersecurity purposes or Educational demonstrations of network load behavior

## Overview
Boykisser is a simulated network load-testing tool built with PyQt5 that emulates HTTP-based request floods toward a target URL using multiple modes and optional proxy support. It includes a user-friendly GUI, built-in proxy scraping,
real-time logging, and progress tracking.
